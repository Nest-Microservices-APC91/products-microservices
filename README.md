# Product Microservices

## Dev

1. Clonar el Repositorio
2. instalar dependencies
3. Crear un archivo `.env` basado en el `.env.template`
4. Ejecutar migracion de prisma `npx prisma migrate dev`
5. Ejecutar `npm run start:dev`
